DB TOOLS - SETUP
================

REQUIREMENTS
------------
- Windows
- Python 3.8+
- MariaDB 10.11 (for Maria scripts)
- PostgreSQL 18 (for Postgres scripts)

NO DEPENDENCIES
---------------
All scripts use stdlib only (tkinter, subprocess, os).
Nothing to pip install.

CONFIGURATION
-------------
Before running, open each script and update the credentials at the top:

  MariaCreate.pyw / MariaDelete.pyw
    ROOT_PASSWORD = "your-mariadb-root-password"
    MARIADB_BIN   = path to your mysql.exe

  PostgresCreate.pyw / PostgresDelete.pyw
    ROOT_PASSWORD = "your-postgres-root-password"
    PSQL_BIN      = path to your psql.exe

Default paths assumed:
  MariaDB    -> C:\Program Files\MariaDB 10.11\bin\mysql.exe
  PostgreSQL -> C:\Program Files\PostgreSQL\18\bin\psql.exe

RUN
---
Double-click any .pyw file to launch (no console window).
Or from terminal:
  python MariaCreate.pyw

WHAT EACH SCRIPT DOES
---------------------
  MariaCreate.pyw    Creates a MariaDB database + user with full privileges
  MariaDelete.pyw    Drops a MariaDB database and its user
  PostgresCreate.pyw Creates a PostgreSQL database + user with full privileges
  PostgresDelete.pyw Drops a PostgreSQL database and its user

NOTE: Database name and username are always the same.

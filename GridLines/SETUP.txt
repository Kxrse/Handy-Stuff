SCREEN LINES — SETUP
====================

REQUIREMENTS
------------
- Windows
- Python 3.8+

INSTALL
-------
1. Download Python from https://python.org
   Make sure to check "Add Python to PATH" during install.

2. Open a terminal in the folder and run:
   pip install -r requirements.txt

3. Run the script:
   python ScreenLines.pyw

   Or just double-click ScreenLines.pyw (no console window).

USAGE
-----
- Monitor  : Select which screen to overlay
- Spacing  : Grid line interval in pixels (10 - 200)
- Opacity  : Overlay transparency (0.05 - 1.0)
- Color    : Type a hex value and press Enter, or click a preset swatch
- Enable   : Show the overlay
- Disable  : Hide the overlay

The overlay is fully click-through, nothing underneath is affected.
Close the control window to exit.
